#  Sistema de Gestão e Cadastro (Módulo TCC)

Este projeto é um módulo fundamental desenvolvido para o meu Trabalho de Conclusão de Curso (TCC) em Desenvolvimento de Sistemas. Ele se concentra na persistência de dados e na interface de usuário para gerenciamento de informações, utilizando a pilha clássica PHP e MySQL.

##  Sobre o Projeto

O objetivo deste módulo é gerenciar o fluxo de entrada de dados do sistema, garantindo a integridade das informações desde a interface (Front-end) até o banco de dados (Back-end). O sistema conta com validações de campos e uma arquitetura pensada para escalabilidade.

##  Principais Funcionalidades

*   **Módulo de Cadastro:** Interface amigável para registro de novos usuários/dados.
*   **Persistência em Banco de Dados:** Integração completa com MySQL.
*   **Arquitetura de Dados:** Planejamento estruturado via DER e Diagramas de Caso de Uso.
*   **Interface Responsiva:** Telas de Login e Cadastro adaptáveis.

##  Tecnologias Utilizadas

*   **Linguagem:** PHP 8.x
*   **Banco de Dados:** MySQL
*   **Front-end:** HTML5, CSS3
*   **Modelagem:** BrModel/Astah

##  Documentação Técnica

Diferente de projetos simples, este sistema foi planejado seguindo boas práticas de engenharia de software. Você pode encontrar os diagramas na pasta `/docs`:

*   **DER (Diagrama Entidade-Relacionamento):** Estrutura lógica do banco de dados.
*   **Diagrama de Caso de Uso:** Representação das interações do usuário com o sistema.

## ⚙️ Como Executar o Projeto

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/Jao-lab51/Sistema-de-Cadastro.git
    ```
2.  Importe o arquivo SQL localizado em `/db/database.sql` para o seu servidor MySQL.
3.  Configure a conexão com o banco no arquivo `config.php`.
4.  Mova a pasta para o diretório `htdocs` do seu servidor local.
