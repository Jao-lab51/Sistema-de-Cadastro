Sistema de Gestão e Cadastro (Módulo TCC)
    Este projeto é um módulo fundamental desenvolvido para o meu Trabalho de Conclusão de Curso (TCC) em Desenvolvimento
    de Sistemas. Ele foca na persistência de dados e na interface de usuário para gestão de informações, utilizando a 
    stack clássica PHP e MySQL.

Sobre o Projeto
    O objetivo deste módulo é gerenciar o fluxo de entrada de dados do sistema, garantindo a integridade das informações
    desde a interface (Front-end) até o banco de dados (Back-end). O sistema conta com validações de campos e uma 
    arquitetura pensada para escalabilidade.

Principais Funcionalidades:
    Módulo de Cadastro: Interface amigável para registro de novos usuários/dados.
    Persistência em Banco de Dados: Integração completa com MySQL.
    Arquitetura de Dados: Planejamento estruturado via DER e Diagramas de Caso de Uso.
    Interface Responsiva: Telas de Login e Cadastro adaptáveis.

Tecnologias Utilizadas
    Linguagem: PHP 8.x
    Banco de Dados: MySQL
    Front-end: HTML5, CSS3 (e/ou Framework utilizado, ex: Bootstrap/Tailwind)
    Modelagem: BrModel/Astah (ou a ferramenta que você usou para o DER)

Documentação Técnica
    Diferente de projetos simples, este sistema foi planejado seguindo boas práticas de engenharia de software. 
    Você pode encontrar os diagramas na pasta /docs:
    DER (Diagrama Entidade-Relacionamento): Estrutura lógica do banco de dados.
    Diagrama de Caso de Uso: Representação das interações do usuário com o sistema.

Como Executar o Projeto
    Clone este repositório:
    git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
    Importe o arquivo SQL localizado em /db/database.sql para o seu servidor MySQL (XAMPP, WAMP, etc ).
    Configure a conexão com o banco no arquivo config.php (ou o nome do seu arquivo de conexão).
    Mova a pasta para o diretório htdocs ou www do seu servidor local.
    Acesse localhost/NOME_DA_PASTA no seu navegador.
